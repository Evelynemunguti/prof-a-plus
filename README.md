# PROF. A+ — Academic & IT Services Website

A modern, responsive one-page website for **PROF. A+**, designed to help students quickly access academic and IT support services through WhatsApp.

---

## About PROF. A+

This project is a **conversion-focused landing page** built to:

* Help students get academic support quickly
* Provide clear information about services offered
* Drive users to contact via WhatsApp instantly

The design is clean, mobile-first and optimized for performance and simplicity.

---

## Features

* Fully responsive (mobile, tablet, desktop)
* Modern UI with brand colors (blue, teal, gold)
* Smooth scrolling navigation
* Scroll-based animations (fade-in effects)
* Sticky navigation bar
* WhatsApp integration (direct chat + floating button)
* Structured service sections

---

## Built With

* HTML5
* CSS3 (Flexbox + Grid)
* JavaScript (Vanilla JS)
